# 🎠 Carousel en JavaScript

## 📖 Description
Ce projet montre comment créer un carousel d'images interactif et responsive en utilisant HTML, CSS et JavaScript.

## 📸 Aperçu
![Aperçu](img/Carrousel%20-%20Google%20Chrome%2016_04_2026%2011_41_24.png)

## ⚙️ Installation
1. Créer trois fichiers :
   - `index.html` → structure
   - `style.css` → design
   - `script.js` → logique du carousel
2. Ouvrir `index.html` dans le navigateur
